Seeing Clearly, Reasoning Confidently

Plug-and-play remedies for rare object reasoning in frozen vision-language models.

Xin Hu, Haomiao Ni, Yunbei Zhang, Jihun Hamm, Zechen Li, Zhengming Ding
Paper arXiv Supplement Code Data Checkpoints Citation

VLMs can reason fluently about the wrong object.

Rare objects in driving scenes are often visible to humans but under-attended by general VLMs. Our method learns class-aware multi-modal embeddings and uses them to refine visual tokens and provide object hints, while keeping the VLM backbone frozen.

Paper Figure 1: the baseline identifies a bollard as a traffic light, while our plug-and-play refinement recovers the correct object and reasoning.

Frozen VLM backbone during adapter training and inference.

One class memory, two plug-and-play remedies.

The framework learns multi-modal class embeddings from region-level visual evidence and synonym-augmented text. The same class memory drives visual token enhancement and object-aware prompt enrichment at inference time.

Paper Figure 3: multi-modal class embeddings support visual token refinement and object-aware prompt enrichment while the VLM remains frozen.

Multi-modal class embeddings separate rare categories.

The learned class memory aligns visual evidence with synonym-augmented text anchors, making rare object categories more separable than raw VFM features alone.

Paper Figure 4: compared with DINOv3 features, our multi-modal class embeddings form cleaner class-level structure.

CODA-LM is the main reproduction path.

The public release centers on CODA-LM region perception because it directly matches the paper's rare object reasoning motivation and provides enough training and test coverage for reproducible experiments.

Interpretable behavior.
